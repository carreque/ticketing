# Design: `scripts/gen_env.py` — populate `.env` from Terraform outputs

## Purpose

After `terraform apply`, regenerate a repo-root `.env` from the live Terraform
outputs so local tooling and tests can point at the deployed stack.

## Flow

1. **Locate the `terraform` binary** — `shutil.which("terraform")`, falling back
   to `C:\Program Files\terraform\terraform.exe` (it is not on PATH in this
   environment), with a `TERRAFORM` env-var override.
2. **Read outputs** — run `terraform output -json` with `cwd` = the `terraform/`
   directory, capture stdout.
3. **Map keys** — parse JSON; for each expected key, map to an UPPER_SNAKE env
   var and pull `.value`:

   | Terraform output      | .env key               |
   | --------------------- | ---------------------- |
   | `api_base_url`        | `API_BASE_URL`         |
   | `user_pool_id`        | `USER_POOL_ID`         |
   | `user_pool_client_id` | `USER_POOL_CLIENT_ID`  |
   | `sns_topic_arn`       | `SNS_TOPIC_ARN`        |
   | `dynamodb_table`      | `DYNAMODB_TABLE`       |

4. **Write `.env`** — write repo-root `.env` (overwrite silently) as plain
   `KEY=value` lines, with a leading comment:
   `# Generated by scripts/gen_env.py — do not edit; re-run after terraform apply`.
5. **Summarize** — print a one-line summary (`Wrote 5 values to .env`).

## Error handling

Fail fast, exit 1 with a clear message:

- terraform binary not found → tell the user to install it or set `TERRAFORM`.
- `terraform output` fails / empty (no state applied yet) →
  "No Terraform outputs found — run `terraform apply` first."
- An expected key missing from outputs → warn which one, still write the rest.

## Dependencies

None beyond the standard library (`subprocess`, `json`, `shutil`, `os`,
`pathlib`). `.env` is already covered by `.gitignore`, so it is never pushed to
git.

## Testing

Per the repo's TDD convention, a `tests/` pytest that monkeypatches the
`terraform output -json` call to return sample JSON and asserts the `.env`
contents — no real AWS or Terraform invocation needed.
